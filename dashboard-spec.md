## 상태 정의

각 항목은 현재 날짜 기준으로 아래 3가지 상태 중 하나를 가진다.

| 상태 | 조건 | 색상 |
|------|------|------|
| 🟢 양호 | 다음 예정일까지 여유 있음 | 초록 |
| 🟡 곧 해야함 | 다음 예정일까지 N일 이하 남음 | 노랑 |

## 화면 구성

### 헤더

- 제목: `🏠 우리집 대시보드`
- 오늘 날짜 표시
- 전체 항목의 상태 요약 칩 (예: 🔴 기간 지남 1개, 🟡 곧 해야함 2개)

### 카드 목록

항목마다 카드 하나. 카드에 표시할 정보:

- 아이콘 + 이름 + 주기
- 상태 배지
- 경과 프로그레스 바 (마지막 완료일 → 다음 예정일 기준)
- 마지막 완료일 / 다음 예정일
- `✓ 오늘 완료` 버튼 — 클릭 시 오늘 날짜로 완료 기록
- `✏️ 수정` 버튼 — 항목 편집 모달 열기

### 항목 추가 버튼

카드 목록 마지막에 `＋ 항목 추가` 버튼.

---

## 인터랙션

- **오늘 완료**: 버튼 클릭 → 마지막 완료일을 오늘로 저장 → 상태 즉시 갱신
- **수정**: 모달에서 이름 / 아이콘 / 주기 / 마지막 완료일 / 알림 기준일 수정 가능
- **추가**: 새 항목 모달에서 작성 후 저장
- **삭제**: 수정 모달 안에서 삭제 가능

---

## 데이터 저장

- 브라우저 `localStorage`에 JSON으로 저장
- 별도 서버/백엔드 없음

---

## 창틀 청소 — 세부 동작

- 카드 하나로 표시, 30일 주기
- 카드 안에 방 목록(서재·아이방·베란다·주방·거실)을 행으로 표시
- 각 방마다 마지막 완료일 및 상태 표시
- 전체 카드 상태는 가장 오래된 방 기준
- `✓ 완료 기록` 버튼 클릭 → 어느 방 청소했는지 체크박스로 선택 → 저장
