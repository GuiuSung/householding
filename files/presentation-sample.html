📤 배포하기

Step 1: 로컬 작업

git clone https://github.com/GuiuSung/kun.log.git
cd kun.log
# 파일 수정...

Step 2: 커밋 및 푸시

git add .
git commit -m "Add new presentation"
git push origin main

Step 3: GitHub Pages 활성화

  • 저장소 Settings → Pages
  • Source: main 브랜치 선택
  • Save 클릭
  • 1-2분 후 배포 완료!

💡 팁 & 트릭

  • 🎨 --r-main-color로 테마 커스터마이징
  • 🖼️ 배경 이미지 추가: data-background="url"
  • 🔗 링크 추가: <a href="url">
  • 📸 이미지 삽입: <img src="url">
  • 🎥 비디오 포함: <video src="url">
